## 1. APRESENTAÇÃO

O **Pesca Lead** é uma plataforma completa de extração, enriquecimento e gestão de leads B2B, desenvolvida para otimizar a prospecção comercial da sua empresa. Com tecnologia de ponta e múltiplas fontes de dados, oferecemos uma solução robusta para capturar leads qualificados e prontos para conversão.

---

## 2. FONTES DE EXTRAÇÃO DE LEADS

### 2.1 Extração CNPJ (Base de Dados Brasileira)

Acesso direto a uma **base com mais de 50 milhões de empresas brasileiras**, atualizada e validada.

**Filtros Disponíveis:**

| Filtro | Descrição |
|--------|-----------|
| **UF (Estado)** | Selecione um ou múltiplos estados |
| **Município** | Filtre por cidade específica |
| **CEP** | Busca por código postal |
| **CNAE** | Atividade econômica principal da empresa |
| **Porte da Empresa** | Micro, Pequena, Média ou Grande empresa |
| **Capital Social** | Faixa de capital social da empresa |
| **Regime Tributário** | Simples Nacional, Lucro Presumido, Lucro Real |
| **Situação Cadastral** | Ativa, Suspensa, Inapta, Baixada |
| **Tipo** | Matriz ou Filial |
| **Data de Abertura** | Período de constituição da empresa |
| **Com E-mail** | Apenas empresas que possuem e-mail cadastrado |
| **Com Telefone** | Apenas empresas que possuem telefone cadastrado |

**Prévia em Tempo Real:** Antes de extrair, o sistema mostra quantos leads correspondem aos seus filtros, incluindo disponibilidade de e-mail e telefone.

---

### 2.2 Extração Google Maps

Captura de leads diretamente do Google Maps com dados comerciais completos.

**Funcionalidades:**

| Recurso | Descrição |
|---------|-----------|
| **Busca por Nicho** | Digite o tipo de negócio (ex: "Clínicas Odontológicas") |
| **Busca por Localização** | Cidade, Estado ou Bairro específico |
| **Expansão Inteligente** | IA que expande automaticamente para bairros e regiões próximas |
| **Filtro por Avaliações** | Mínimo de avaliações no Google |
| **Filtro por Nota** | Nota mínima de avaliação (1-5 estrelas) |
| **Exigir Website** | Apenas empresas com site cadastrado |
| **Exigir Telefone** | Apenas empresas com telefone |
| **Exigir E-mail** | Apenas empresas com e-mail disponível |
| **Busca Estadual** | Expande a busca para todo o estado |
| **Deduplicação Automática** | Remove resultados duplicados automaticamente |

---

### 2.3 Extração Instagram

Sistema de descoberta e enriquecimento de perfis comerciais do Instagram.

**Processo em 3 Fases:**

1. **Fase de Descoberta**
   - Busca perfis por nicho + localização
   - Identificação de perfis comerciais

2. **Fase de Enriquecimento**
   - Extração completa dos dados do perfil
   - Processamento em lotes de até 100 perfis simultâneos

3. **Fase de Migração**
   - Conversão para leads qualificados
   - Integração automática ao funil de vendas

**Dados Extraídos do Instagram:**

| Campo | Descrição |
|-------|-----------|
| **Username** | Nome de usuário do perfil |
| **Nome Completo** | Nome exibido no perfil |
| **Biografia** | Descrição completa do perfil |
| **Seguidores** | Quantidade de seguidores |
| **Seguindo** | Quantidade de contas seguidas |
| **Publicações** | Número de posts |
| **Verificado** | Status de verificação (selo azul) |
| **Conta Comercial** | Se é conta business |
| **Categoria** | Categoria do negócio |
| **E-mail** | E-mail do perfil ou extraído da bio |
| **Telefone** | Telefone do perfil ou extraído da bio |
| **WhatsApp** | Link do WhatsApp extraído da bio |
| **Website** | URL externa do perfil |

## 4. ENRIQUECIMENTO INTELIGENTE DE DADOS

O Pesca Lead possui um **Orquestrador de Enriquecimento** que automaticamente complementa as informações dos seus leads.

### 4.1 Pipeline de Enriquecimento

**Lead Capturado** > **Validação WhatsApp** > **WHOIS** > **CNPJ** > **Lead Enriquecido**

### 4.2 Enriquecimento por WhatsApp

| Processo | Descrição |
|----------|-----------|
| **Normalização** | Padroniza o número para formato brasileiro |
| **Validação** | Verifica se o número possui WhatsApp ativo |
| **Conta Business** | Identifica se é WhatsApp Business |
| **Resultado** | Campo `whatsapp_valid` atualizado automaticamente |

### 4.3 Enriquecimento por WHOIS

Para leads com website, consultamos o registro de domínio:

| Dado | Descrição |
|------|-----------|
| **Domínio** | URL do site |
| **Registrante** | Dados do proprietário do domínio |
| **CNPJ do Registrante** | CNPJ associado ao domínio (para .br) |

### 4.4 Enriquecimento por CNPJ

Quando identificamos um CNPJ (via WHOIS ou extração direta):

| Dado Enriquecido | Fonte |
|------------------|-------|
| **Dados Cadastrais Completos** | Base CNPJ 50M+ empresas |
| **Sócios** | Quadro societário |
| **Atividades Secundárias** | CNAEs secundários |
| **Histórico de Situação** | Alterações cadastrais |
| **E-mails Adicionais** | E-mails secundários da empresa |
| **Telefones Adicionais** | Telefones secundários |

### 4.5 Consolidação de Contatos

O sistema consolida automaticamente todos os contatos encontrados:

**Telefones:**
